    Phoenix-ui of React

    phoenix-ui是基于React框架实现的移动端UI组件库。

    github: https://github.com/future-team/phoenix-ui

    特性

    • 有较丰富的组件可供选择,也将持续丰富和优化
    • 组件具有拓展性

    安装

    组件中使用到动画的部分需要额外安装 react-addons-css-transition-group 来支持

        npm install phoenix-ui --save
    npm install react-addons-css-transition-group --save

    使用

    CommonJS

        var Button = require('phoenix-ui/lib/Button')

    ES6

        import {Button} from 'phoenix-ui'

    Browser globals

        <script src="http://future-team.github.io/phoenix-ui/dist/phoenix-ui.min.js"></script>

    其他

    phoenix-ui需要和 phoenix-styles 配合使用,提供阿波罗默认样式phoenix-styles.min.css,需要IOS样式同时引入skin-ios.min.css即可。
