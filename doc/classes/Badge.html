    Badge Class

    Badge徽章标记，主要是用来提供不同颜色的标识
    可以通过phStyle来改变颜色
    因为发生回调的概率不大所以没有提供回调函数
    但支持自定义类名，事件等操作
    具体属性和接口如下：

    • phStyle:badge徽章颜色,可取值[primary、warning、danger、info、error、success]等，默认primary
    • Badge phStyle="success" 惠2 /Badge
